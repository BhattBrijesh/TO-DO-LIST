# TO-DO-LIST

A simple and efficient To-Do List web application built with **PHP** and **MySQL**. This project allows users to manage daily tasks by creating, updating, completing, and deleting tasks through a clean and user-friendly interface.

## 🚀 Features

* Add New Tasks
* Edit Tasks
* Mark Tasks as Completed
* Delete Tasks
* Task Status Management
* Simple Dashboard
* Responsive Layout
* Easy-to-Use Interface

## 🛠️ Tech Stack

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap

## ⚙️ Installation

1. Copy the project folder to your local server (`htdocs` for XAMPP / `www` for WAMP)

2. Import the database into phpMyAdmin

3. Update database connection settings

4. Start Apache and MySQL

5. Open browser:

```text id="d3x7yw"
http://localhost/TO-DO-LIST
```

## ▶️ Main Functions

* Create Tasks
* Update Tasks
* Complete Tasks
* Delete Tasks
* Organize Daily Work
